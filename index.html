¿Qué aprenderás?

  • HTML: El lenguaje de Marcado de Hipertexto te permitirá
    generar la estructura de una página web.

  • CSS: El lenguaje de Estilo en Cascada te dará la posibilidad
    de darle personalización y diseño a los elementos de tu página web.

  • JAVASCRIPT: Este lenguaje de Programación te permitirá darle
    interacción y movimiento a tu página web.

  • PYTHON: Este lenguaje de Programación te dará la posibilidad de
    desarrollar la parte analítica de tu pagina web y conectarla a
    una base de datos.

  • SQL: El lenguaje de Programación de bases de datos por excelencia.
    Con el aprenderás administrar todos los datos de tu página web.
